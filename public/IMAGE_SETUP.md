# Image Setup Instructions

To complete the landing page setup, please add the following images to the `/public` folder:

1. **qr-menu-stand.jpg** - Image of a QR code menu stand on a restaurant table
   - This image will be used in the "How It Works" section, Step 1
   - Recommended size: 800x600px or similar aspect ratio
   - Should show a QR code menu stand on a table in a restaurant setting

2. **qr-scanning.jpg** - Image of a person scanning a QR code with their smartphone
   - This image will be used in the "How It Works" section, Step 2
   - Recommended size: 800x600px or similar aspect ratio
   - Should show someone holding a phone and scanning a QR code

The images you provided in the conversation should be saved with these exact filenames:
- `qr-menu-stand.jpg` (or .png)
- `qr-scanning.jpg` (or .png)

Once added, the landing page will automatically display them in the "How It Works" section.
